Home
====

This site provides articulation mapping files generated from the [ArtiCluster](https://github.com/r-koubou/articluster) definition files managed on [ArticulationMappingFiles](https://github.com/r-koubou/ArticulationMappingFiles), which are compatible with various DAWs.

**This site is automatically updated via GitHub Actions.**

## Supported Formats

Following files format supported.

- Cakewalk Sonar
    - Articulation map file (*.artmap)
- Cubase / Nuendo
    - VST Expression map (*.expressionmap)
- Logic Pro
    - Articulation Set file (*.plist)
- Studio One / Studio Pro
    - Sound Variation file (*.keyswitch)

## Download Individual Files

You can download individual files from the navigation bar at the top of the screen.

## Download as ZIP File

If you'd like to download everything as a single ZIP file, you can do so from the [GitHub releases page](https://github.com/r-koubou/ArticulationMappingFiles/releases/latest).


## GitHub Repository

https://github.com/r-koubou/ArticulationMappingFiles


## Contribution

We have also released a tool that allows you to convert between different articulation mapping file formats. This tool is available on GitHub at [ArtiCluster](https://github.com/r-koubou/articluster).

If you create a file in this tool’s format (which we’ve named “Universal Definition”) and submit a pull request, it will be added to this site.
