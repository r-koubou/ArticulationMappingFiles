Cubase / Nuendo
===============

You will probably have the following folder, so put the file down after you download it.

## Cubase

- Windows: `C:\Users\[USERNAME]\AppData\Roaming\Steinberg\Cubase [VERSION]\Presets\Expression Maps`
- macOS: `/Users/<USERNAME>/Library/Preferences/Steinberg/Cubase [VERSION]/Presets/Expression Maps`

## Nuendo

- Windows: `C:\Users\[USERNAME]\AppData\Roaming\Steinberg\Nuendo [VERSION]\Presets\Expression Maps`
- macOS: `/Users/[USERNAME]/Library/Preferences/Steinberg/Nuendo [VERSION]/Presets/Expression Maps`
